## View slides locally

First, ensure you have the following installed:

1. [Node.js](http://nodejs.org)

Then, install dependencies and run the preview server:

```bash
$ npm install
$ npm start
```

When you are done, you can deploy to gh-pages by running:
```bash
$ npm run deploy
```
